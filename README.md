### Installation
#### Virtual Environment [recommended]
##### Dependencies
```sh
$ sudo apt install python3-venv
```
##### Run venv
```sh
$ . venv/bin/activate
(venv)$ python -V
Python 3.6.5
$ python run.py
```
#### Base OS
```sh
$ pip install -r requirements.txt
$ python3 run.py
```
